### Usage

Run conduktor platform :    
Replace `<your license key>` with the license provided
```sh
./run-local.sh "<your license key>"
```

Stop conduktor platform
```sh
./stop-local.sh
```

If you want to delete all conduktor platform data
```sh
./rm-local.sh
```
